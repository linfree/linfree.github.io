2017-05-20 20:07

一、安装Elasticsearch-Head

1.插件安装方式（推荐）

在Elasticsearch目录下

如果提示

是因为好像2.0以上的版本-install 变成了 install了。

2.下载安装方式

从https://github.com/mobz/elasticsearch-head下载ZIP包。

二、重启Elasticsearch。访问。
访问地址是： http://{你的ip地址}:9200/_plugin/head/

http 端口默认是: 9200 。
